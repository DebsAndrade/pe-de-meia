# Pé de Meia 🧦

Aplicacao web em React para apoiar o planejamento financeiro e o habito de economizar com foco no futuro.

## Tecnologias

- React 19
- Vite 8
- ESLint 9

## Requisitos

- Node.js 20+ (recomendado)
- pnpm (recomendado) ou npm

## Como executar localmente

1. Instale as dependencias:

```bash
pnpm install
```

2. Rode o projeto em modo de desenvolvimento:

```bash
pnpm dev
```

3. Abra no navegador:

```text
http://localhost:5173
```

## Scripts disponiveis

- `pnpm dev`: inicia servidor de desenvolvimento com hot reload.
- `pnpm build`: gera build de producao.
- `pnpm preview`: executa preview local da build de producao.
- `pnpm lint`: roda verificacao de padroes com ESLint.

Se preferir npm, substitua `pnpm` por `npm run` (ex.: `npm run dev`).
